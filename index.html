Remember how chaotic Twitch Plays Pokemon was? Now we have a much harder challenge: install Arch Linux. Every five seconds, the most popular keystroke in Twitch chat will be entered into an Arch Linux virtual machine.

Twitch in The Shell @ Arch Linux has no rules (except don't do anything illegal).

  • Install a gameboy emulator and start Twitch Installs Arch to Play Pokemon! - Completed 11/03/2015
  • Install and run memetest! - Completed 11/03/2015
  • Install and play Dwarf Fortress! - Completed 11/04/2015

  • Jailbreak the VM! - Bonus points for this one :D
  • Play a movie in the terminal and within X!
